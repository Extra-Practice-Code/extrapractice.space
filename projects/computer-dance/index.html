Computer Dance (exquisite coding / against the clockuter) *

A warm up coding exercise that occurred on May 17, 2022 at XP.

* https://nl.wikipedia.org/wiki/Stoelendans

Ingredients

  • floor (big table optional)
  • 3+ friends
  • 3+ laptops installed with code editors

Instructions

  1. everyone sits in a circle.
  2. start the timer.
  3. when the countdown ends the timer will provide a prompt (switch computers, continue coding, or take a break).
  4. go for a few rounds and when it feels right share the websites you've created with one another.
